## ✨ **Special Effects**

### **1. Shadows**
```dart
BoxShadow(
  color: Colors.black.withOpacity(0.15),
  blurRadius: 20,
  offset: Offset(0, 10),
)
```

### **2. Gradients**
```dart
// Header
LinearGradient(colors: [Black → Dark])

// Buttons
LinearGradient(colors: [Black → Dark])

// Icons
LinearGradient(colors: [Red → Red Dark])
```

### **3. Accent Stripes**
```dart
Container(
  width: 4,
  height: 16,
  gradient: Red to Red Dark,
  borderRadius: 2,
)
```
